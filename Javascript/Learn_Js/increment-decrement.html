Increment and Decrement

                
Ever wonder what the difference between i++ and ++i was? Did you know both were options? i++ returns i and then
                    increments it whereas ++i increments i and then returns it.

                
                    
let i = 0;
                        console.log(i++);
                        // 0
                        let i = 0;
                        console.log(++i);
                        // 1
